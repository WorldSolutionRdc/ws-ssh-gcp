| ⚡ **Type de proxy** | V2Ray WebSocket |

---

## 🛠️ **Déploiement**

```bash
gcloud run deploy v2ray-tunnel \
  --source . \
  --platform managed \
  --region europe-west2 \
  --allow-unauthenticated \
  --port 8080 \
  --memory 512Mi \
  --cpu 1 \
  --timeout 3600
```
